# 🏢 Real Estate CRM (MERN Stack)

A full-stack Real Estate CRM application built using MERN stack to manage leads, properties, clients, deals, and follow-ups efficiently.

---

## 🚀 Features

### 🔑 Lead Management
- Add, update, delete leads
- Lead status tracking (New, Contacted, Qualified, Closed, Lost)
- Search & filter leads
- Convert lead to client

### 🏠 Property Management
- Add & manage property listings
- Property details (price, location, size, amenities)
- Property status tracking

### 👤 Client Management
- Create and manage clients
- Lead → Client conversion
- Client data tracking

### 💼 Deal Management
- Create deals
- Link clients and properties
- Deal status tracking
- Commission tracking

### 🔔 Follow-up System
- Add follow-up date
- View today's follow-ups

### 📊 Dashboard
- Overview of system data (Leads, Clients, Deals)

---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Atlas)  
- **Authentication:** JWT
